Features

Everything you need for personal cloud

FileRoll combines a robust WebDAV server with a native desktop sync client, giving you full control over your files.

WebDAV Server

Standards-compliant WebDAV with chunked upload support. Works with any WebDAV client.

Desktop Sync

Native client for Windows, macOS, and Linux. Two-way, upload-only, or download-only sync modes.

Multi-User

Built-in user management with per-user storage quotas and isolated file spaces.

Link Sharing

Generate shareable links with optional password protection and expiration dates.

Version Control

Automatic file versioning lets you browse and restore previous versions at any time.

Self-Hosted & Private

Your data stays on your server. No third-party tracking, no vendor lock-in. Fully open source.

Quick Start

Up and running in minutes

Deploy the server, install the client, start syncing.

1

Deploy the Server

Clone the repo and point your web server, or use Docker for a one-command setup.

docker run -d -p 80:80 fileroll/server
2

Create an Account

Open the web UI, sign up, and your personal cloud space is ready.

https://your-server.example.com
3

Install & Sync

Download the desktop client, enter your server URL and credentials, and sync begins automatically.

fileroll-sync --server https://your-server
